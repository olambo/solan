# Solan

Four independent tracks for constructing a multi-world, multi-substrate civilisation: species insurance at Ceres, the Human-Vero substrate transition, Solan sovereignty, and the void economy.

Part of the [Kaisune](https://github.com/olambo/kaisune) corpus. Published at [olambo.github.io/kaisune/solan](https://olambo.github.io/kaisune/solan/).

## Structure

Content lives in `solan/` — a Hugo content module consumed by Kaisune.
